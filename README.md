# Bank Management System (Java)

A simple **Banking Management System** built in Java using object‑oriented programming.  
This CLI project simulates basic banking operations such as creating accounts, depositing, withdrawing, and viewing transactions.

---

##  Features
- Create new bank accounts (Savings & Current)
- Deposit money
- Withdraw money
- View account details
- Record and view transactions

---

## Technologies Used
- Java (Core Java)
- Object‑Oriented Programming (OOP)
- Command Line Interface (CLI)
